Studio Guide

What is TSP Art?

TSP Art transforms your images into a single, continuous line drawing. It uses the Travelling Salesperson Problem (TSP)—a classic mathematical challenge—to find the shortest possible path that visits thousands of points generated from your image's dark areas.

The Algorithm

  1. Stippling: Your image is processed into thousands of points (stipples) based on local density.
  2. Greedy Skeleton: A fast initial path is built using a nearest-neighbor approach.
  3. k-opt Untangling: A sophisticated Variable-Depth Search (Lin-Kernighan inspired) iteratively removes edge crossings, shortening and smoothing the final line.

Tunable Parameters

  • Point Density: Controls how many points are generated. More points mean more detail but slower processing.
  • Optimization Depth: An exponential multiplier (0.1x to 1000x) that determines the search budget for untangling. Higher depth yields a cleaner, more optimal path.
  • Optimization Breadth: Determines the spatial search radius for finding intersections. Balanced (25%) is usually sufficient.
